## Project Overview
This project focuses on building a reliable system to monitor temperature from multiple sources using **LM35 sensors** and an **Arduino Mega 2560**. The data is processed and outputted in JSON format for easy integration with other systems.

## Key Features
* **Multi-channel Sensing:** Supports multiple LM35 sensors simultaneously.
* **Data Formatting:** Real-time temperature data output in JSON format.
* **Version Control:** Managed entirely using Git and hosted on GitHub.

## Technical Specifications
* **Hardware:** Arduino Mega 2560, LM35 Temperature Sensors.
* **Software:** Arduino IDE, C/C++.
* **Protocol:** Serial Communication (UART).
